Thoughts

A passion project made during the Covid pandemic

It's a remaster of a game I made in 2018

There were supposed to be more levels and the bigger dragons were meant for harder enemies

But as I leared the hard way, the scope was WAY too ambitious

I learned that animating each enemy frame-by-frame is a huge time commitment

I put a lot of effort into this so I wanted to wrap it up into a game

the goal was to learn new Unity features

Bonus

After wrapping the game up I felt like redesigning the dragon again and making concepts for the enemies I had in mind
